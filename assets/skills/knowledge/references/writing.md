# Writing Memory

## Overview

Writing memory stores writing style guidelines that must be applied when creating documentation.

## Guidelines

| Rule                | Value                   |
| ------------------- | ----------------------- |
| Language            | English only            |
| Max Sentence Length | 25 words                |
| Max List Items      | 4 items                 |
| Tone                | Professional, objective |

## Usage

```python
@omni("knowledge.get_writing_memory")
```

## Related Commands

- `get_writing_memory` - Load writing rules
- `writer.lint_writing_style` - Check text against guidelines
- `writer.check_markdown_structure` - Validate markdown format
