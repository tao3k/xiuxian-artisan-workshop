# Formatting Standards

## Markdown Conventions

### Headings

- Use `#` for title, `##` for major sections, `###` for subsections
- No more than 3 levels of nesting
- Add one blank line before and after headings

### Lists

- Use `-` for unordered lists
- Use `1.` for ordered lists (auto-numbering)
- Max 4 items per list; use subheadings if more

### Code Blocks

- Language tag required: `python, `bash, etc.
- No hard wraps in code blocks
- Comment complex lines inline

### Tables

- Use tables for 2-4 related items
- Left-align text, right-align numbers
- Truncate long values with "..." if needed
